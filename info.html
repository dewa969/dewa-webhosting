Kelebihan Dedicated/Colocation Server 1.Satu Server fisik yang hanya dipergunakan untuk Anda. 2.memiliki akses root server. 3.Anda yang menerapkan aturan sendiri. 4.Kontrol penuh untuk meningkatkan spesifikasi server sendiri. 5.Resource yang besar untuk Anda sendiri. Kekurangan Dedicated/Colocation Server 1.Diperlukan pengetahuan dasar tentang server dan sistem operasi. 2.Biaya yang dikeluarkan lebih tinggi dari shared hosting. 3.Harus menginstall control panel sendiri. 4.Jika melakukan upgrade/downgrade maka akan mengalami downtime karena server harus dimatikan.
