Prinsip Kerja Share Hosting Server Prinsip Kerja Shared Hosting (atau biasa disebut web hosting saja) memberi peluang menggunakan satu server yang terhubung ke internet dan sedang digunakan bersama-sama dengan pengguna hosting lainnya. Semua website pelanggan akan berjalan pada sistem operasi dan control panel yang sama. Dalam hal ini, sebuah server dapat dipakai bersama-sama oleh ratusan hingga ribuan website. Namun demikian, seseorang tidak akan tahu siapa atau situs webapa yang digunakan untuk berbagi sumber daya server karena setiap pelanggan memilki batasan pada jumlah total sumber daya server yang dapat digunakan. Dengan demikian shared hosting identik dengan server di web hosting satu ditempati oleh banyak anggota domain/p>
